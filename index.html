Hướng dẫn kiểm tra

XAMPP và Laravel
  1. Mở XAMPP, bật Apache và MySQL.
  2. Vào phpMyAdmin và import file database-schema.sql.
  3. Nếu chạy Laravel, cài Composer rồi chạy migrate và seed theo README.
  4. Kiểm tra endpoint /api/kiem-tra-du-lieu sau khi backend chạy.
